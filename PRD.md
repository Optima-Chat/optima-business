## 1. 项目概述

### 1.1 项目背景
極致智能科技有限公司（Optima AI Limited）需要一个专业的企业官网，用于展示 AI 技术定制开发服务能力，吸引潜在企业客户，建立技术品牌形象。

### 1.2 项目目标
- **业务目标**: 获取企业级技术定制开发项目线索
- **品牌目标**: 建立"AI全栈技术服务专家"的专业形象
- **转化目标**: 访客 → 咨询 → 商务洽谈 → 项目签约

### 1.3 核心价值主张
**"AI 驱动的全栈技术服务，从算法到产品，端到端交付"**

---

## 2. 产品定位

### 2.1 目标客户
- **主要客户**: 有 AI/ML 技术需求的企业（TO B）
- **客户特征**:
  - 传统企业数字化转型
  - 互联网公司 AI 能力升级
  - 创业公司技术外包需求
- **决策人**: CTO、技术负责人、产品负责人

### 2.2 差异化定位
| 维度 | 传统外包公司 | AI咨询公司 | **Optima AI** |
|-----|------------|-----------|--------------|
| 技术深度 | ⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 交付能力 | ⭐⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐⭐⭐ |
| AI前沿技术 | ⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 全栈能力 | ⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐⭐⭐ |

**核心优势**:
- ✅ 顶级团队（腾讯/阿里/港科大）
- ✅ 深度 AI 能力（CLIP、FAISS、MUSIQ 等前沿技术）
- ✅ 全栈交付（前端到算法，端到端）

## 4. 技术方案

### 4.1 技术栈
```yaml
框架: Next.js 14 (App Router)
语言: TypeScript
样式: Tailwind CSS
组件库: shadcn/ui
动画: Framer Motion
表单: React Hook Form + Zod
图标: Lucide React
部署: Vercel
域名: optima.business 或 business.optima.shop
```

### 4.2 项目结构
```
optima-business/
├── app/
│   ├── layout.tsx              # 全局布局
│   ├── page.tsx                # 首页
│   ├── cases/
│   │   └── page.tsx            # 案例页
│   └── contact/
│       └── page.tsx            # 联系页
├── components/
│   ├── hero.tsx                # Hero Section
│   ├── core-capabilities.tsx  # 核心能力卡片
│   ├── business-scope.tsx     # 业务范围
│   ├── case-card.tsx          # 项目卡片
│   ├── navbar.tsx             # 导航栏
│   └── footer.tsx             # 页脚
├── lib/
│   └── cases-data.ts          # 案例数据
├── public/
│   └── images/                # 图片资源
└── styles/
    └── globals.css            # 全局样式
```

## 5. 设计规范

### 5.1 色彩方案
```css
/* 主色调 - 深蓝/科技紫 */
--primary: #6366f1        /* Indigo 500 */
--primary-dark: #4f46e5   /* Indigo 600 */
--primary-light: #818cf8  /* Indigo 400 */

/* 辅助色 - 渐变 */
--gradient: linear-gradient(135deg, #6366f1 0%, #8b5cf6 100%)

/* 中性色 */
--background: #0a0a0a     /* 深色背景 */
--surface: #1a1a1a        /* 卡片背景 */
--border: #2a2a2a         /* 边框 */
--text: #ffffff           /* 主文字 */
--text-muted: #a0a0a0     /* 次要文字 */

/* 强调色 */
--accent: #10b981         /* Green 500 - 成功/数据 */
--warning: #f59e0b        /* Amber 500 */
```

### 5.2 字体系统
```css
/* 标题 */
font-family: 'Inter', 'PingFang SC', sans-serif

/* 正文 */
font-family: 'Inter', 'PingFang SC', sans-serif

/* 代码 */
font-family: 'Fira Code', 'JetBrains Mono', monospace
```

### 5.3 间距系统（Tailwind 默认）
- 4px 基准单位
- 常用间距: 4, 8, 12, 16, 24, 32, 48, 64px

### 5.4 组件规范

#### 按钮
- 主按钮: 渐变背景 + 白色文字
- 次按钮: 透明背景 + 边框 + 主色文字
- 尺寸: sm / md / lg

#### 卡片
- 背景: 深色表面色（#1a1a1a）
- 边框: 1px 半透明边框
- 圆角: 12px
- 阴影: 微弱外发光（可选）

#### 图标
- 尺寸: 24px / 32px / 48px
- 颜色: 主色或白色
- 风格: 线性图标（Lucide React）

## 6. 内容策略

### 6.1 文案原则
- **专业性**: 技术准确，避免夸大
- **简洁性**: 核心信息优先，避免冗余
- **可信度**: 用数据说话（80倍提升、2个月交付）

### 6.2 案例匿名化原则
- ✅ 可展示: 行业、技术栈、成果数据、项目规模
