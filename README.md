# Optima AI - 商业定制开发官网

極致智能科技有限公司（Optima AI Limited）的企业官网，展示 AI 技术定制开发服务能力。

## 项目定位

面向企业客户的 AI 技术定制开发服务官网，重点展示：
- 深度 AI 技术能力（计算机视觉、NLP、推荐系统）
- 全栈开发能力（从算法到产品，端到端交付）
- 成功案例（匿名化展示）
- 团队实力（腾讯/阿里/港科大背景）

## 技术栈

- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **组件**: shadcn/ui
- **动画**: Framer Motion
- **部署**: Vercel

## 文档

- [产品需求文档 (PRD)](./PRD.md) - 完整的产品需求和设计规范

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

访问 http://localhost:3000 查看网站。
