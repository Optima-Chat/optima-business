## 十、关键规律

### 1. 所有案例都是"大企业自建/重度定制"

平安有自研大模型平台、美的部署了 5000+ Agent、波司登有 AI 实验室、飞鹤建了 AI 中台——投入门槛极高，中小企业无法复制。

### 2. 三个最具普适性的场景

| 场景 | 出现频率 | 代表案例 |
|------|----------|----------|
| 自然语言查数据 / 辅助决策 | 几乎所有 | 一汽 GPT-BI、招行"招小财"、顺丰"丰知"、瓴羊分析 Agent |
| 行业知识编码为 Agent | 高频 | 绝味 14.3 万条知识库、格创东智 35,000+ 工业模型 |
| 供应链 / 库存智能决策 | 高频 | 波司登滚动下单、京东物流超脑、蒙牛交付缩短 55% |

### 3. 核心公式已被验证

```
企业数据接入 + 行业知识编码 + 自然语言交互 = 可量化的降本增效
```

这正是 Optima Pulse 概念文档中提出的架构：

```
CLI 工具层（数据接入） + Skills 层（行业知识） + Agent 层（自然语言交互）
```

区别在于：上述案例全部面向大企业，Optima Pulse 的机会在于**用 SaaS 模式把同样的能力带给中小企业**。
