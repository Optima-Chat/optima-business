# 智谱AI 竞争格局

> 调研日期：2026-01-27

## 中国AI大模型"六小虎"格局演变

### 2023年：并驾齐驱

零一万物、百川智能、智谱AI、MiniMax、月之暗面、阶跃星辰并称"AI六小虎"，DeepSeek与面壁智能为"6+2"格局中的"2"。

### 2025年：明显分化

| 公司 | 战略方向 | 当前状态 | 备注 |
|------|----------|----------|------|
| **智谱AI** | AGI全栈 + MaaS + ToB | 港股上市，第一梯队 | 全球大模型第一股 |
| **MiniMax** | C端社交（Talkie/星野） | 准备港股IPO，第一梯队 | 海外C端强势 |
| **月之暗面** | Kimi + OK Computer | 商业化承压 | 削减投流，推出OK Computer目标1亿美元收入 |
| **百川智能** | 医疗垂类 | 退出通用模型竞争 | 停止超大规模通用模型训练 |
| **零一万物** | AI应用层 | 转型 | 放弃超大模型训练，聚焦商业化应用 |
| **阶跃星辰** | 多模态 | 持续迭代 | - |
| **DeepSeek** | 开源 + 推理 | 异军突起 | R1引爆全球 |

## 智谱 vs 大厂

### 价格战

2025年，字节跳动（豆包）、百度（文心一言）、阿里（通义千问）发起惨烈价格战，将API调用价格压到"地板价"甚至免费。

| 厂商 | 策略 | 对智谱的影响 |
|------|------|-------------|
| 字节（豆包） | API极低价 | 直接冲击MaaS收入 |
| 阿里（通义） | 配合新模型积极降价 | 挤压中小客户 |
| 百度（文心） | 补贴获客 | 抢占政企市场 |
| 月之暗面 | 多款模型最高降83.3% | 加剧行业内卷 |
| 智谱自身 | 代表性模型降价超90% | 被动跟随 |

### 智谱的差异化优势

1. **独立第三方定位** — 不与客户竞争（大厂本身是客户的竞对）
2. **自主可控** — 清华系背景，全国产化芯片适配，政务客户天然偏好
3. **本地化部署能力** — 大厂云端服务为主，难以下沉做私有化
4. **中立生态** — 不绑定任何云平台

## 核心竞争壁垒分析

### 智谱的护城河

| 壁垒 | 强度 | 说明 |
|------|------|------|
| 技术底蕴 | 强 | 清华系，自研GLM架构，持续迭代 |
| 自主可控 | 强 | 40+国产芯片适配，实体清单下仍可运作 |
| 政企客户信任 | 强 | 清华背景+国资股东+中立定位 |
| 开源生态 | 中 | 3000万下载，但DeepSeek冲击 |
| 开发者社区 | 中 | GLM Coding Plan爆发但需持续投入 |
| 价格竞争力 | 弱 | 大厂可以亏本补贴，智谱资金有限 |

### 面临的挑战

1. **巨额亏损**：62亿累计亏损，短期盈利无望
2. **芯片管制**：实体清单限制高端算力获取
3. **算力瓶颈**：需求增长快于算力扩容
4. **价格战**：大厂将API价格压到地板价
5. **DeepSeek冲击**：开源推理模型R1引爆全球，分流关注度
